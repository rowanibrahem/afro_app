# Afro Asian Board App

Educational Flutter Project

<h2 align="left"> <img src="https://github.com/urfavcrow23/QuizApp/blob/main/Images/1_cGaTxrPGlA29Z9QcaRg-xA.gif?raw=true" width="30">  Features </h2>

1. Authentication
2. Home Screen that displays available courses based on their tags
3. Show course details
4. Integrate with PayPal to allow payment from the course
5. Ability to Add to favorite and Add to cart
